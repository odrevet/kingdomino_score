    <h2>Top menu buttons</h2>
    <ul>
        <li>Puzzle shape: Activate / Deactivate the Age of Giants or Lacour extension</li>
        <li>Shield: Select / Unselect quests
            <ul>
                <li>Two quests maximum can be selected at a time.</li>
                <li>There are more quests when Age of Giants is activated</li>
                <li>The number of quests activated are displayed in a red bubble</li>
            </ul>
        </li>
        <li>5 / 7: change the size of the board. Changing the size of the board resets all tiles</li>
        <li>Trash: Reset the board</li>
        <li>About: Display author and license</li>
    </ul>

    <h2>Score</h2>
    <p>The score is updated as each change on the board.</p>
    <p>Tapping the score display will cycle to calculations details, pie per land type and total</p>

    <h2 id="bottom-menu-buttons">Tiles bar</h2>
    <p>Each land type has its own color:</p>
    <ul>
        <li>Yellow: Wheat</li>
        <li>Light green: Grassland</li>
        <li>Dark green: Forest</li>
        <li>Water: Lake</li>
        <li>Grey: Swamp</li>
        <li>Brown: Mine</li>
        <li>Blue grey: Empty</li>
        <li>Castle: Place your castle</li>
    </ul>

    <ul>
        <li>Crown: Place / remove crowns</li>
        <li>Giant (when AoG is activated): Place / remove giants</li>
    </ul>
